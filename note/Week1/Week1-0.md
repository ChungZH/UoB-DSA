## ADT/抽象数据类型

Aka Abstract Data Type

- a type
- with associated operations
- whose representation is hidden to the user

## Memory Model/内存模型

OS++ 系统存在内存 Mem[-]

使用如下操作控制 OS++：

- `allocate_memory(n)`: 分配 `n` 个未使用的内存
- `free_memory(address, n)`: 释放自 `address` 起 `n` 个内存
